# Gestion des Réclamations - Structure de base

Ce projet contient une structure complète en PHP natif avec séparation par rôles :
- Utilisateur
- Agent
- Administrateur

La connexion à la base Oracle est prévue via ODBC dans `php/database.php`.
Vous pouvez adapter les modèles et les contrôleurs en fonction de votre MLD réel.
